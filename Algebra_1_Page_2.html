Linear Equation In Two Variables

A variable is a letter or symbol that represents the various values that a variable could be within mathematical operations. A common variable used in algebra is x and y. Since we are evaluating linear equations, the variable is written with no exponent or root function. The purpose of this linear equations in two variables is to provide knowledge on how the dependent variable (y) changes with respect to the independent variable (x).
